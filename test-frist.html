index.pl

This is the main controller which bootstraps the application. It uses the template TEMPLATES/start.htm which loads the various required parts of the application.

This controller sends a ClientConf object to the template renderer (as do almost all controllers).

This controller handles much of the issues related to billing for Details clients as well as displaying service-wide client notices from the admin.client_notices table.

This controller has many legacy functions which have been deprecated but not removed from the controller, but also code related to the V2 of Details, both of which are not of concern to Believe.
